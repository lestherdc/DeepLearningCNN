# Deep Learning Model

Implementacion de modelo deep learning usando CNN + LSTM con
mecanismo de atencion, el algoritmo reconocera patrones y nos
dara idea de prediccion si el precio de la accion tocara minimo o
maximo del dia anterior, o se quedara en rango del mismo dia entre
su maximo o minimo.

## Libreias necesarias

- tensorflow
- tensorflow.keras
- panda
- skylearn
